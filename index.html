Qu'est-ce que CrossBlocks ?

CrossBlocks est un wargame tactique à base de LEGO. Créez vos propres fiches d'unités avec des statistiques personnalisées (points de vie, armure, armes, points d'action), simulez des batailles stratégiques et imprimez vos fiches au format A4. Utilisez vos minifigurines LEGO comme unités de combat !

L'éditeur en ligne est 100% gratuit — pas d'inscription, pas de téléchargement. Que vous jouiez avec des minifigures LEGO, des figurines peintes ou du papier, CrossBlocks s'adapte à votre style de jeu. Les règles complètes du jeu sont disponibles sur notre serveur Discord.

Comment jouer ?
  1. Rejoignez le Discord — lisez les règles complètes et posez vos questions à la communauté
  2. Créez vos unités — utilisez l'éditeur ci-dessus pour définir les caractéristiques de chaque figurine
  3. Imprimez vos fiches — exportez en PDF A4 pour avoir vos fiches pendant la partie
  4. Composez votre armée — choisissez vos unités avec un budget en points
  5. Jouez ! — déployez vos LEGO sur un plateau et affrontez vos amis

Transfert de données entre ordinateurs

Pourquoi pas de sauvegarde directe sur GitHub ? GitHub Pages est un hébergement statique en lecture seule — les visiteurs ne peuvent pas écrire de fichiers sur le serveur. Pour un stockage en ligne partagé, il faudrait un serveur backend (Firebase, Supabase, etc.). En attendant, utilisez Export/Import JSON pour transférer les données entre ordinateurs.

Workflow : Sur le PC où les utilisateurs créent → l'admin exporte le JSON. Sur un autre PC → l'admin importe le JSON. Les données sont fusionnées automatiquement (pas d'écrasement).

Rééquilibrage — Simulation Monte Carlo

Système intelligent : chaque unité est testée en combat simulé (150 batailles) contre des Clones Phase 2 à budget égal. Le prix est ajusté pour que le taux de victoire soit ~50%. Max ±30% de changement par unité.
